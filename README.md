# **Anti Chat Clear**
> This Addon ensures you can see all messages in chat even if the chat is cleared.

### Installation
1. Press `Win` + `R`
2. Paste this into the window that popped up: `%appdata%/.minecraft/LabyMod/addons-1.12` and press enter
3. It should open your Labymod addon directory; Paste the [Anti_Chat_Clear.jar](https://github.com/RappyLabyAddons/AntiChatClear/releases/download/1.0.0/Anti_Chat_Clear.jar) in there.
4. Launch your Labymod client.

### Config
- "Enabled" If the addon is enabled or not
